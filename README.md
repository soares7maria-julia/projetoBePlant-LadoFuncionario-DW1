# Projeto BePlant-3Bi
09/2025

 RELATO PESSOAL DA DISCIPLINA DE DW1
  
Sobre meu projeto: 

Meu projeto do 3° e 4° bimestre  é uma loja online de plantas, denominada por mim de BePlant. Meu projeto todo é dividido em dois repositórios: Um repositório armazena o lado que o cliente vê (ProjetoBePlant-LadoCliente-3Bi-DW1), e o outro armazena o lado que o funcionário vê ( projetoBePlant-LadoFuncionario-DW1). 


Como foi a minha experiência com o uso de IA: 

Esse ano usamos IA para fazer nossos projetos, e a maioria das coisas que conseguimos fazer foi graças a ela.
 Eu tive uma boa relação com a IA, nao tive muitos problemas com o uso dela, é claro, as vezes ela fazia tudo errado, estragava algo que já estava funcionando ou nao entendia o que eu queria, mas temos que lembrar que ela é uma inteligência ARTIFICIAL e por isso temos que ter uma certa paciência e sempre escrever detalhadamente o que você quer que ela faça, imagino que esse foi nosso erro muitas vezes, não detalhar para ela exatamente o que você quer.


Uso da IA pontos positivos e negativos: 

Ponto Positivo: 

Com a IA conseguimos fazer muitas coisas, que, sem ela, seria quase impossível de fazer em um ano considerando nosso nível de conhecimento em programação.
A inteligência artificial é muito boa em fazer css bonito…frontend no geral.


Ponto Negativo:

É difícil lidar com IA quando estamos começando no mundo da programação, principalmente porque normalmente não sabemos o que ela está fazendo. 
Não é tão boa assim com backend quanto com frontend.  


Aprendizados:

• Organizar os arquivos de projetos;
• Consegui ter uma boa noção de como funciona um site;
• Entendi o que cada arquivo do meu projeto faz;
• entendi na prática a diferença entre backend e frontend;
• Aprendi a subir o servidor;
• entendi de certa forma como conectar um projeto com o banco de dados;
• O básico do github, fazer um repositório, conectar ao projeto, fazer commit; 
•  Usar a IA de forma funcional;
• etc.

Considerações finais: 

Este ano foi ótimo para mim, aprendi e consegui fazer muitas coisas, bem mais do que eu imaginava que ia conseguir fazer. 
Fiquei satisfeita com o que consegui entregar no meu projeto, poderia ter feito melhor, entregado os projetos antes do prazo, mas estou contente com o entreguei. 

E por fim, minha parte favorita entre o frontend e o backend foi… o backend.
